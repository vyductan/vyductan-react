# Form Internal Components

Internal subcomponents that make up the form structure.

## Files and Directories

| Path                           | What                                        | When to read                           |
| ------------------------------ | ------------------------------------------- | -------------------------------------- |
| `index.tsx`                    | Component exports                           | Accessing internal form components     |
| `form-control.tsx`             | Form control wrapper component              | Understanding form field rendering     |
| `form-item-control-input.tsx`  | Input control for form items                | Managing input field state             |
| `form-field.tsx`               | Form field component with validation        | Building validated form fields         |
| `form-item-row.tsx`            | Row layout component for form items         | Creating horizontal form layouts       |
| `form-label.tsx`               | Label component for form fields             | Rendering form field labels            |
| `form-root.tsx`                | Root form container component               | Setting up form structure              |
| `form-item.tsx`                | Form item wrapper with label and validation | Creating form field groups             |
| `form-message.tsx`             | Validation message component                | Displaying field-level errors          |
| `form-list.tsx`                | Dynamic form list component for arrays      | Implementing dynamic field arrays      |
| `form-errors-notification.tsx` | Notification component for form errors      | Showing form-level error notifications |
| `form-error-list.tsx`          | List component for multiple errors          | Displaying all form validation errors  |
| `form-description.tsx`         | Description/help text component             | Adding helper text to fields           |
