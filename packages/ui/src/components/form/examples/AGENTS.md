Example implementations demonstrating form usage patterns.

| Path                  | What                                              | When to read                                       |
| --------------------- | ------------------------------------------------- | -------------------------------------------------- |
| `basic.tsx`           | Basic composable form with schema-required labels | Getting started with shadcn-style form composition |
| `antd-basic.tsx`      | Previous basic FormItem implementation            | Checking legacy FormItem behavior                  |
| `shadcn.tsx`          | Form with shadcn/ui styling                       | Using shadcn-style form components                 |
| `layout-multiple.tsx` | Form with multiple layout options                 | Implementing complex form layouts                  |
| `antd.tsx`            | Form with Ant Design styling                      | Using Ant Design-style forms                       |
