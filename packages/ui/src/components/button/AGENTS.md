# Button Component

Button component with various styles, sizes, and states.

## Files

| What                                         | When to read                                |
| -------------------------------------------- | ------------------------------------------- |
| `button-variants.ts` - Button variant styles | Customizing button styles, adding variants  |
| `button.mdx` - Button documentation          | Learning button API, usage guidelines       |
| `button.stories.tsx` - Button stories        | Viewing button examples, testing variations |
| `button.tsx` - Main button component         | Implementing buttons, click handlers        |
| `index.tsx` - Button exports                 | Using button component, importing button    |
| `loading-icon.tsx` - Button loading icon     | Implementing loading states, spinner icons  |

## Subdirectories

| What                      | When to read                            |
| ------------------------- | --------------------------------------- |
| `demo/` - Button examples | Learning button usage, viewing examples |
