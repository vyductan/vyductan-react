# TypeScript Configuration

Shared TypeScript compiler configuration presets for the workspace.

## Navigation

| File                     | Description                      | When to read                                 |
| :----------------------- | :------------------------------- | :------------------------------------------- |
| `base.json`              | Base TypeScript compiler options | adjusting shared TS compiler settings        |
| `compiled-package.json`  | Compiled package config          | reviewing build output configuration         |
| `package.json`           | Package configuration            | managing dependencies or package metadata    |
