# ESLint Configuration

Shared ESLint configuration presets for the workspace.

## Navigation

| File            | Description                     | When to read                                    |
| :-------------- | :------------------------------ | :---------------------------------------------- |
| `base.ts`       | Base ESLint rules               | adjusting core linting rules for all projects   |
| `nextjs.ts`     | Next.js specific rules          | configuring linting for Next.js applications    |
| `react.ts`      | React specific rules            | configuring linting for React projects          |
| `package.json`  | Package configuration           | managing dependencies or package metadata       |
| `tsconfig.json` | TypeScript compiler config      | adjusting type checking for config files        |
