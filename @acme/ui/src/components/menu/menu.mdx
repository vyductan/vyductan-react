# Menu

A menu component that displays a list of options. Supports vertical, horizontal, and inline modes with nested sub-menus, groups, and dividers. Built following Ant Design API patterns with Shadcn UI styling.

## Examples

### Nested Menu

Menu supports nested sub-menus with groups and cascading menus. You can create complex menu structures with multiple levels of nesting.
