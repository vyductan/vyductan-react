# Editor Component

Rich text editor built on Lexical framework with extensive plugin architecture, custom nodes, and theming support.

## Files and Directories

| Path                   | What                                        | When to read                                   |
| ---------------------- | ------------------------------------------- | ---------------------------------------------- |
| `editor.tsx`           | Main editor component implementation        | Implementing rich text editing features        |
| `editor.stories.tsx`   | Storybook stories for editor component      | Testing editor in different scenarios          |
| `editor-providers.tsx` | Provider components for editor context      | Setting up editor context and state management |
| `types.ts`             | TypeScript type definitions for editor      | Understanding editor API and prop types        |
| `index.ts`             | Public exports for editor module            | Importing editor components and utilities      |
| `ROADMAP.md`           | Future features and development plans       | Planning editor enhancements                   |
| `components/`          | Reusable UI components for editor           | Building custom editor UI elements             |
| `context/`             | React context providers for editor state    | Managing editor context and shared state       |
| `demo/`                | Example implementations and demos           | Learning editor usage patterns                 |
| `editor-hooks/`        | Custom React hooks for editor functionality | Implementing editor behavior and state logic   |
| `editor-ui/`           | UI components for editor interface          | Customizing editor visual elements             |
| `nodes/`               | Custom Lexical nodes for content types      | Adding new content types to editor             |
| `plugins/`             | Plugin implementations for editor features  | Extending editor functionality                 |
| `shared/`              | Shared utilities and helpers                | Accessing common editor utilities              |
| `themes/`              | Theme configurations for editor styling     | Customizing editor appearance                  |
| `transformers/`        | Content transformers for import/export      | Converting content between formats             |
| `utils/`               | Utility functions for editor operations     | Using helper functions for editor tasks        |
