# Dropdown Component

Dropdown menu component with support for nested menus, positioning, and various trigger options.

## Files and Directories

| Path              | What                                   | When to read                          |
| ----------------- | -------------------------------------- | ------------------------------------- |
| `dropdown.tsx`    | Main dropdown component implementation | Implementing dropdown menus           |
| `_components.tsx` | Internal dropdown subcomponents        | Understanding dropdown menu structure |
| `index.ts`        | Public exports for dropdown module     | Importing dropdown components         |
| `demo/`           | Example dropdown implementations       | Learning dropdown usage patterns      |
