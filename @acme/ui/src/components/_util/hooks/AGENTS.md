# Utility Hooks

This directory contains shared React hooks used across UI components.

## Files

| What                                               | When to read                                      |
| -------------------------------------------------- | ------------------------------------------------- |
| `use-force-update.ts` - Force update hook          | Forcing component re-renders, manual updates      |
| `use-multiple-select.ts` - Multiple selection hook | Implementing multi-select, selection state        |
| `use-sync-state.ts` - Synchronized state hook      | Syncing state, controlled/uncontrolled components |
