# Table Filter Hooks

Hooks and components for table filtering functionality.

## Files and Directories

| Path                  | What                            | When to read                     |
| --------------------- | ------------------------------- | -------------------------------- |
| `index.tsx`           | Main filter hook and logic      | Implementing table filtering     |
| `filter-search.tsx`   | Search-based filter component   | Adding search filters to columns |
| `filter-dropdown.tsx` | Dropdown-based filter component | Creating dropdown filter menus   |
