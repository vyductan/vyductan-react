# @acme/ui/.storybook

Storybook configuration for the UI component library.

## Files

| File              | What                                           | When to read                                    |
| ----------------- | ---------------------------------------------- | ----------------------------------------------- |
| `main.ts`         | Main Storybook configuration                   | Configuring Storybook addons, stories, features |
| `preview.tsx`     | Storybook preview configuration and decorators | Setting up global styles, providers, themes     |
| `vitest.setup.ts` | Vitest setup for Storybook interaction tests   | Configuring test environment                    |
| `main.ts.backup`  | Backup of previous Storybook configuration     | Rolling back configuration changes              |
