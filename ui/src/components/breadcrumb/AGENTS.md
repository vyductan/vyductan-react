# Breadcrumb Component

Navigation breadcrumbs for hierarchical navigation.

## Files

| What                                         | When to read                                        |
| -------------------------------------------- | --------------------------------------------------- |
| `breadcrumb.tsx` - Main breadcrumb component | Implementing breadcrumbs, navigation trails         |
| `index.ts` - Breadcrumb exports              | Using breadcrumb component, importing breadcrumb    |
| `use-breadcrumb.tsx` - Breadcrumb hook       | Managing breadcrumb state, programmatic breadcrumbs |

## Subdirectories

| What                                            | When to read                                                |
| ----------------------------------------------- | ----------------------------------------------------------- |
| `_components/` - Breadcrumb internal components | Understanding breadcrumb structure, customizing breadcrumbs |
