# Utility Functions and Helpers

This directory contains shared utility functions and hooks used across UI components.

## Files

| What                                             | When to read                                        |
| ------------------------------------------------ | --------------------------------------------------- |
| `easings.ts` - Easing functions                  | Implementing animations, transitions                |
| `extends-object.ts` - Object extension utilities | Extending objects, merging configurations           |
| `get-scroll.ts` - Scroll position utilities      | Getting scroll positions, scroll handling           |
| `media-query-util.ts` - Media query utilities    | Working with responsive breakpoints, media queries  |
| `responsive-observer.ts` - Responsive observer   | Implementing responsive behavior, viewport tracking |
| `scroll-to.ts` - Scroll utilities                | Programmatic scrolling, smooth scrolling            |
| `type.ts` - Type utilities                       | Working with TypeScript types, type helpers         |
| `use-scroll.ts` - Scroll hook                    | Using scroll tracking, scroll events                |
| `warning.ts` - Warning utilities                 | Displaying warnings, development alerts             |
| `z-index.context.ts` - Z-index context           | Managing z-index layers, stacking contexts          |

## Subdirectories

| What                          | When to read                      |
| ----------------------------- | --------------------------------- |
| `hooks/` - Shared React hooks | Using utility hooks, custom hooks |
