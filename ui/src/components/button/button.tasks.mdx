# Button Component - Tasks & Planning

## Overview
Button component với API tương tự Ant Design, hỗ trợ nhiều variants, sizes, và states khác nhau.

> 📚 **Documentation**: Xem examples và API chi tiết tại `button.docs.mdx`

## Current Implementation Status

### ✅ Completed Features
- [x] Basic button rendering with variants (default, primary, ghost, outline, etc.)
- [x] Size variants (small, middle, large, icon, sm mapping)
- [x] Loading state with spinner
- [x] Disabled state support
- [x] Icon support (prefix/suffix)
- [x] Link integration (href prop)
- [x] AsChild pattern for composition
- [x] Wave effect animation
- [x] Color variants (default, danger, success, link)
- [x] Shape variants (default, circle, round)
- [x] Screen reader only text (srOnly)
- [x] HTML type support (submit, reset, button)
- [x] Tailwind variants configuration
- [x] Custom classNames support
- [x] TypeScript type safety
