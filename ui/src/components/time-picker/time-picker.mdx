## API Reference

### TimePicker Props

| Property       | Type                                        | Default         | Description                         |
| -------------- | ------------------------------------------- | --------------- | ----------------------------------- |
| `value`        | `Dayjs`                                     | -               | The current time value              |
| `defaultValue` | `Dayjs`                                     | -               | The default time value              |
| `onChange`     | `(time: Dayjs, timeString: string) => void` | -               | Callback when time changes          |
| `format`       | `string`                                    | `"HH:mm"`       | The time format                     |
| `showNow`      | `boolean`                                   | -               | Whether to show the "Now" button    |
| `disabled`     | `boolean`                                   | `false`         | Whether the time picker is disabled |
| `placeholder`  | `string`                                    | `"Select time"` | Placeholder text                    |
| `status`       | `"error" \| "warning"`                      | -               | Validation status                   |
