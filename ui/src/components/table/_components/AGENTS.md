# Table Internal Components

Internal subcomponents that make up the table structure.

| Path                        | What                                       | When to read                           |

| `table-toolbar-left.tsx`    | Left section of table toolbar              | Adding actions to toolbar left side    |

| `table-toolbar-root.tsx`    | Root toolbar container component           | Customizing table toolbar structure    |
| `table-summary-context.tsx` | Context for summary row state              | Managing summary row data              |
| `view-options.tsx`          | Column visibility toggle component         | Implementing column show/hide controls |
| `table-head-advanced.tsx`   | Advanced table header with filters/sorting | Building complex table headers         |
| `table-sortable-row.tsx`    | Draggable row component for reordering     | Implementing row drag-and-drop         |
| `table-summary.tsx`         | Summary row container component            | Creating table footer summaries        |
| `table-toolbar-right.tsx`   | Right section of table toolbar             | Adding toolbar actions to right side   |
| `table-header.tsx`          | Standard table header component            | Rendering table column headers         |
| `base.tsx`                  | Base table implementation                  | Understanding core table rendering     |
| `expand-icon.tsx`           | Icon component for expandable rows         | Customizing expand/collapse icons      |
| `fixed-holder.tsx`          | Component for fixed columns                | Implementing sticky columns            |
| `table-summary-row.tsx`     | Row component for summary section          | Building summary row structure         |
