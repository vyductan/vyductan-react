# Table Hooks

Custom React hooks for table functionality and state management.

## Files and Directories

| Path                   | What                                           | When to read                          |
| ---------------------- | ---------------------------------------------- | ------------------------------------- |
| `use-table.tsx`        | Main hook for table state and configuration    | Implementing custom table logic       |
| `use-pingnings.ts`     | Hook for detecting scroll position and pinning | Adding fixed column scroll effects    |
| `use-lazy-kv-map.ts`   | Hook for lazy-loading key-value mappings       | Optimizing large dataset performance  |
| `use-columns.tsx`      | Hook for column configuration and rendering    | Managing dynamic columns              |
| `use-frame.ts`         | Hook for animation frame handling              | Implementing smooth animations        |
| `use-expand.tsx`       | Hook for row expansion state                   | Managing expandable rows              |
| `use-pagination.ts`    | Hook for pagination state and controls         | Adding pagination to tables           |
| `use-filter/`          | Filtering-related hooks and components         | Implementing column filters           |
| `use-sorter.tsx`       | Hook for sorting state and logic               | Adding sorting functionality          |
| `use-selection.tsx`    | Hook for row selection state                   | Implementing row selection checkboxes |
| `use-title-columns.ts` | Hook for generating column titles              | Managing column header rendering      |
