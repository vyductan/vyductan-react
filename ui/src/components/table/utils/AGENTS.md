# Table Utilities

Utility functions for table operations and data manipulation.

## Files and Directories

| Path              | What                                                 | When to read                            |
| ----------------- | ---------------------------------------------------- | --------------------------------------- |
| `legacy-util.ts`  | Legacy utility functions for backwards compatibility | Maintaining compatibility with old code |
| `value-utils.ts`  | Utilities for processing cell values                 | Formatting and transforming cell data   |
| `expand-util.tsx` | Utilities for expandable row handling                | Managing row expansion logic            |
| `fix-util.ts`     | Utilities for fixed columns                          | Implementing sticky column calculations |
