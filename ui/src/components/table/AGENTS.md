# Table Component

Advanced data table component with filtering, sorting, pagination, row selection, and expandable rows.

## Files and Directories

| Path                | What                                   | When to read                                    |
| ------------------- | -------------------------------------- | ----------------------------------------------- |
| `table.tsx`         | Main table component implementation    | Implementing data tables with advanced features |
| `table.stories.tsx` | Storybook stories for table component  | Testing table in different scenarios            |
| `index.tsx`         | Public exports for table module        | Importing table components and utilities        |
| `types.ts`          | TypeScript type definitions for table  | Understanding table API and column definitions  |
| `util.tsx`          | General utility functions for table    | Using table helper functions                    |
| `styles.ts`         | Styling utilities and CSS-in-JS        | Customizing table appearance                    |
| `constant.ts`       | Constants and configuration values     | Accessing default table settings                |
| `table.mdx`         | Documentation for table component      | Learning table usage and API                    |
| `_components/`      | Internal table subcomponents           | Understanding table internal structure          |
| `demo/`             | Example table implementations          | Learning table usage patterns                   |
| `hooks/`            | Custom hooks for table functionality   | Implementing table features with hooks          |
| `locale/`           | Internationalization and locale files  | Adding multi-language support                   |
| `utils/`            | Utility functions for table operations | Using specialized table utilities               |
