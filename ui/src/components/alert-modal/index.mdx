# AlertModal

A confirmation dialog component that displays an alert message with confirm/cancel actions. Built on top of the AlertDialog primitive components and follows Ant Design API patterns with Shadcn UI styling.

### Types

AlertModal supports different types to display different visual styles and icons: `confirm`, `warning`, `info`, `success`, and `error`.

## Layout 2 Cột

Bạn có thể sử dụng `Row` và `Col` để tạo layout 2 cột. Mỗi `Col` với `span={12}` sẽ chiếm một nửa chiều rộng (24 là tổng số cột).

### Sử dụng Row và Col

```tsx
<Row gutter={16}>
  <Col span={12}>
    <div>Nội dung cột 1</div>
  </Col>
  <Col span={12}>
    <div>Nội dung cột 2</div>
  </Col>
</Row>
```
