# Examples

## Basic card

A basic card containing a title, content and an extra corner content. Supports two sizes: `default` and `small`.

```tsx
import { Card } from "@/components/ui/card";

<Card
  title="Default size card"
  extra={<a href="#">More</a>}
  style={{ width: 300 }}
>
  <p>Card content</p>
  <p>Card content</p>
  <p>Card content</p>
</Card>;
```
