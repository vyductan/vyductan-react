### Custom dropdown

Customize the dropdown menu via dropdownRender. If you want to close the dropdown after clicking the custom content, you need to control open prop

```tsx
<Select
  dropdownRender={(menu) => {
    return (
      <>
        {menu}
        <Divider className="my-2" />
        <ButtonAdd
          className="w-full pl-6"
          variant="ghost"
          primary={false}
          srOnly={false}
          onClick={() => {
            setChooseEarningTypeToIndex(index);
          }}
        >
          {`${t("add")} ${t("salary.earning.type")}`}
        </ButtonAdd>
      </>
    );
  }}
/>
```
