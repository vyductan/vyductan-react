## API

### Props

| Property         | Description                                    | Type                         | Default     |
| ---------------- | ---------------------------------------------- | ---------------------------- | ----------- |
| `checked`        | Whether the switch is checked                  | `boolean`                    | `false`     |
| `value`          | Alias for `checked` prop                       | `boolean`                    | -           |
| `onChange`       | Callback when switch state changes             | `(checked: boolean) => void` | -           |
| `className`      | Additional class name for the switch container | `string`                     | -           |
| `disabled`       | Whether the switch is disabled                 | `boolean`                    | `false`     |
| `defaultChecked` | Whether the switch is checked by default       | `boolean`                    | `false`     |
| `size`           | Size of the switch                             | `"small" \| "default"`       | `"default"` |
| `loading`        | Loading state of switch                        | `boolean`                    | `false`     |

### Notes

- Component hỗ trợ cả API của Ant Design (sử dụng `onChange`, `checked`, `value`) và Shadcn Switch (sử dụng `onCheckedChange`)
- `value` prop có độ ưu tiên cao hơn `checked` nếu cả hai đều được cung cấp
