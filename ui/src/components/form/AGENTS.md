# Form Component

Flexible form component with validation, multiple layouts, and integration with form libraries.

## Files and Directories

| Path           | What                                  | When to read                           |
| -------------- | ------------------------------------- | -------------------------------------- |
| `form.tsx`     | Main form component implementation    | Implementing forms with validation     |
| `form.mdx`     | Documentation for form component      | Learning form usage and API            |
| `index.tsx`    | Public exports for form module        | Importing form components and hooks    |
| `types.ts`     | TypeScript type definitions for form  | Understanding form API and field types |
| `utils.ts`     | Utility functions for form operations | Using form helper functions            |
| `_components/` | Internal form subcomponents           | Understanding form field structure     |
| `context/`     | React context for form state          | Managing form context and shared state |
| `demo/`        | Example form implementations          | Learning form usage patterns           |
| `hooks/`       | Custom hooks for form functionality   | Implementing form validation and state |
