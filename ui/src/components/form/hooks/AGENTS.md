# Form Hooks

Custom React hooks for form functionality and validation.

## Files and Directories

| Path                             | What                                            | When to read                            |
| -------------------------------- | ----------------------------------------------- | --------------------------------------- |
| `use-extra-props.ts`             | Hook for handling extra form field props        | Adding custom properties to form fields |
| `use-variant.ts`                 | Hook for form variant styling                   | Managing form visual variants           |
| `use-form.ts`                    | Main hook for form state management             | Implementing custom form logic          |
| `use-field-optionality-check.ts` | Hook for checking field required/optional state | Validating field requirements           |
| `use-form-instance.ts`           | Hook for accessing form instance                | Getting form context and methods        |
