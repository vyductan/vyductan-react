## API Reference

### Form Props

| Property     | Type                                     | Default      | Description                          |
| ------------ | ---------------------------------------- | ------------ | ------------------------------------ |
| `form`       | `UseFormReturn`                          | -            | The form instance from `useForm`     |
| `layout`     | `"horizontal" \| "vertical" \| "inline"` | `"vertical"` | Form layout                          |
| `labelCol`   | `ColProps`                               | -            | Label layout configuration           |
| `wrapperCol` | `ColProps`                               | -            | Control wrapper layout configuration |
| `labelAlign` | `"left" \| "right"`                      | `"right"`    | Label text alignment                 |
| `labelWrap`  | `boolean`                                | `false`      | Whether label can wrap               |
| `colon`      | `boolean`                                | `true`       | Whether to show colon after label    |
| `disabled`   | `boolean`                                | `false`      | Whether to disable all form fields   |

### Field Props

| Property      | Type        | Default | Description                                    |
| ------------- | ----------- | ------- | ---------------------------------------------- |
| `control`     | `Control`   | -       | The control object from `useForm`              |
| `name`        | `string`    | -       | The field name                                 |
| `label`       | `ReactNode` | -       | The field label                                |
| `description` | `ReactNode` | -       | The field description                          |
| `tooltip`     | `ReactNode` | -       | Tooltip text or node                           |
| `required`    | `boolean`   | `false` | Whether the field is required (shows asterisk) |

### useForm

We use `react-hook-form` under the hood. See [react-hook-form documentation](https://react-hook-form.com/docs/useform) for full API details.
