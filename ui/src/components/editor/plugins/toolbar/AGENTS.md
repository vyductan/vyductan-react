# Editor Toolbar Plugins

Plugins and components for editor toolbars.

## Files and Directories

| Path                           | What                                     | When to read                                 |
| ------------------------------ | ---------------------------------------- | -------------------------------------------- |
| `font-size-toolbar-plugin.tsx` | Plugin for font size selector in toolbar | Adding font size controls                    |
| `block-format/`                | Block formatting toolbar components      | Implementing block-level formatting controls |
| Other toolbar plugins          | Various toolbar features                 | Building custom toolbar functionality        |
