# Editor Embed Nodes

Nodes for embedding third-party content from social media and external services.

## Files and Directories

| Path                 | What                           | When to read                           |
| -------------------- | ------------------------------ | -------------------------------------- |
| `instagram-node.tsx` | Node for Instagram post embeds | Embedding Instagram content            |
| `youtube-node.tsx`   | Node for YouTube video embeds  | Embedding YouTube videos               |
| `tiktok-node.tsx`    | Node for TikTok video embeds   | Embedding TikTok content               |
| `twitter-node.tsx`   | Node for Twitter/X post embeds | Embedding Twitter/X posts              |
| `figma-node.tsx`     | Node for Figma design embeds   | Embedding Figma prototypes and designs |
