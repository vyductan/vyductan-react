# Editor Context

React context providers for managing editor state and shared functionality.

## Files and Directories

| Path                              | What                                                        | When to read                                  |
| --------------------------------- | ----------------------------------------------------------- | --------------------------------------------- |
| `image-resolver-context.tsx`      | Context for resolving image URLs and handling image uploads | Implementing image upload functionality       |
| `toolbar-context.tsx`             | Context for toolbar state and actions                       | Managing toolbar visibility and state         |
| `component-picker-context.tsx`    | Context for component picker dropdown                       | Adding slash commands and component insertion |
| `floating-link-context.tsx`       | Context for floating link editor state                      | Implementing link editing functionality       |
| `shared-autocomplete-context.tsx` | Shared context for autocomplete features                    | Building autocomplete and suggestion features |
