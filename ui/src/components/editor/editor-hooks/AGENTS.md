# Editor Hooks

Custom React hooks for editor functionality and state management.

## Files and Directories

| Path                    | What                                                   | When to read                           |
| ----------------------- | ------------------------------------------------------ | -------------------------------------- |
| `use-block-type.ts`     | Hook for managing block type state and transformations | Implementing block formatting features |
| `use-modal.tsx`         | Hook for managing modal dialogs in editor              | Adding modal-based features to editor  |
| `use-suspense-image.ts` | Hook for loading images with Suspense support          | Handling async image loading in editor |
| `use-update-toolbar.ts` | Hook for synchronizing toolbar with editor state       | Updating toolbar based on selection    |
| `use-report.ts`         | Hook for reporting editor events and analytics         | Tracking editor usage and events       |
