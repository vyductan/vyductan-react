# Prettier Configuration

Shared Prettier formatting configuration for the workspace.

## Navigation

| File            | Description                | When to read                              |
| :-------------- | :------------------------- | :---------------------------------------- |
| `index.js`      | Prettier formatting rules  | adjusting code formatting rules           |
| `package.json`  | Package configuration      | managing dependencies or package metadata |
| `tsconfig.json` | TypeScript compiler config | adjusting type checking for config files  |
